目录和文件操作

cd：切换目录

pwd：显示当前目录

mkdir：创建新目录

rmdir：删除空目录

ls

  • -a : 列出全部的文件 包括隐藏档(开头为.的文件)
  • -d：仅列出目录本身
  • -l：列出文件的属性和权限等数据

cp：复制

rm：删除

mv：移动

文件查阅

cat

  • -n：打印出行号
  • -b：列出行号 但是空白行跳过

head -n number fileName ：取出前几行

tail -n number fileName：取出后几行

more和less可以一行一行翻阅

touch：创建文件

file fileName：查看文件类型

指令与文件搜寻

which：搜寻指令的文件名

whereis：在特定目录中查找文件名

locate：根据/var/lib/mlocate内的数据库的记载，找出用户输入的关键词(keyword)文件名。默认一天更新一次数据库

  • updated：手动更新，根据/etc/updatedb.conf的设定去搜寻系统硬盘内的文件名，并更新mlocate内的数据库文件。
